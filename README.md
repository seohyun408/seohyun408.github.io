# seohyun.github.io

GitHub에 **저장소 이름을 정확히 `seohyun408.github.io`**로 두고, 이 저장소 내용을 `main` 브랜치 루트에 푸시하면 **https://seohyun408.github.io** 로 접속할 때 이 사이트가 열립니다.

## 배포 (GitHub Pages)

1. GitHub 저장소 **Settings → Pages**
2. **Build and deployment**: Source를 **Deploy from a branch** 로 두고, Branch는 **main** / **/(root)** 선택 후 Save
3. 몇 분 뒤 위 주소로 접속

## 사진

`assets/images/` 에 JPG·PNG 등을 넣고, HTML의 `img` 태그 `src`에 경로를 적으면 됩니다. (예: `assets/images/profile.jpg`)
